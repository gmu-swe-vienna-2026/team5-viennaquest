## Branching Convention

Use a branch per user story, referencing your Jira story ID:

```bash
git checkout -b feature/PROJ-7-search-endpoint
git commit -m "PROJ-7: add search filter logic"
# Open a pull request to main when the story meets its acceptance criteria
```

## Jira

Each commit message must reference the Jira story ID (e.g., `PROJ-14: implement login screen`).
